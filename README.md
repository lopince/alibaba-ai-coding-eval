# Alibaba AI Coding Evaluation

This repository contains evaluation tasks and benchmarks for Alibaba's AI coding product assessment.

## Overview

This project is designed to systematically evaluate AI coding assistants across various programming tasks, including:

- Code generation and completion
- Bug detection and fixing
- Code refactoring and optimization
- Test case generation
- Documentation generation

## Structure

```
.
├── tasks/              # Evaluation tasks and prompts
├── solutions/          # Reference solutions and expected outputs
├── evaluators/         # Scoring and evaluation scripts
├── datasets/           # Test datasets and code samples
└── results/            # Evaluation results and reports
```

## Getting Started

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run evaluations: `python run_eval.py`

## Contributing

Add new tasks to the `tasks/` directory following the existing format.
