Section 4

Benchmark

Comprehensive benchmark comparison across four model series — Anthropic Claude (Opus 4.6/4.7/4.8, Sonnet 4.6), OpenAI (GPT-5 / GPT-5.5), Qwen (3.7 Max/Plus, 3.6 Max/Plus), and Chinese Frontier (DeepSeek V4 Pro/Flash, GLM-5.1, Kimi K2.6, MiniMax M3, MiMo V2.5 Pro) — evaluated on five benchmarks:

  • LiveBench — General capability (reasoning, coding, agentic) with monthly-refreshed questions to prevent contamination
  • SWE-bench Verified + SWE-bench Pro — Real-world GitHub issue resolution and Scale AI’s harder stricter variant
  • LiveCodeBench v6 — Contamination-free competitive programming
  • Terminal-Bench 2.0/2.1 — Terminal agent tasks
  • Kilo Bench — Real-world developer task completion rate and cost

4.1  General Capability — LiveBench

LiveBench evaluates reasoning, coding, math, data analysis, language, and instruction following with monthly-refreshed questions to prevent data contamination. Scores below from the high-unseen-bias leaderboard (livebench.ai). — = not present in dataset.

General Capability: Anthropic Opus models lead reasoning (88–90), with Opus 4.8 topping at 89.71. GPT-5.5 edges ahead on coding (82.47). Qwen3.7-Max is the strongest Qwen entry (83.34 reasoning). Among Chinese Frontier, DeepSeek V4 Pro leads reasoning (82.69) while Kimi K2.6 is strong across coding (78.57) and agentic (58.33). MiniMax M3 uniquely supports Text + Image + Video + Audio multimodality.

Key observations:
• Qwen3.7-Max remains the strongest Qwen entry on coding (SWE-bench Verified 80.4%, LiveCodeBench 91.6); Qwen3.7-Plus, Qwen3.6-Max, and Qwen3.6-Plus cluster tightly in the 77–79% SWE-V range with strong LiveCodeBench scores (87–90).
• SWE-bench scores vary dramatically by scaffold — always note the evaluation setup. GPT-5.5 SWE-V ranges 82–88% across different scaffolds; DeepSeek V4 self-reported 83.7% but NIST CAISI independently scored it at 74%.
• SWE-bench Pro is much harder — top scores range 38–69% vs. 55–87% on Verified. Claude Opus 4.8 leads Pro at 69.2%; Claude Opus 4.7 scored 64.3%.
• GPT-5.5 leads agent benchmarks (82.7% Terminal-Bench, 74.2% Kilo Bench).
• MiMo V2.5 Pro ($4.92/attempt) and MiniMax M3 ($10.35) offer the best cost-efficiency among Chinese frontier models on Kilo Bench.
