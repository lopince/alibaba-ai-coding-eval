Cost Analysis

Cost comparison for a ~100-member team based in Singapore (infra, coding, data analysis, management) across the current AWS Bedrock + Claude Code setup and proposed AliCloud Bailian / Qoder alternatives. Chinese model pricing shown at China mainland rates; Bailian international pricing (Singapore region) is ~1.5–1.7× higher but the cost advantage remains significant. Frontier Chinese models: Qwen 3.6/3.7, DeepSeek V4, GLM-5.1, Kimi K2.6, MiniMax M3.

Important caveats:
• Singapore international pricing: Bailian per-token rates in the Singapore region are ~1.5–1.7× China mainland prices. The cost advantage remains significant (3–10× vs Opus) but verify actual rates in the Bailian console.
• Bailian Coding Plan model roster is older-gen: Current $50/mo tier includes Qwen3.5-Plus, Qwen3-Max, Qwen3-Coder-Next/Plus, GLM-5, GLM-4.7, Kimi K2.5, MiniMax M2.5 — not the latest Qwen3.7-Plus, DeepSeek V4, Kimi K2.6, or MiniMax M3 listed elsewhere in this page. Verify included models in the Coding Plan docs.
• MiniMax M3 on Bailian: The Bailian model catalog currently lists MiniMax M2.7, not M3. The ¥2.1/¥8.4 row uses MiniMax's official API rate (~$0.30/$1.20) converted to RMB; confirm Bailian availability before depending on it.
• Claude Code Max subscription caps are shared between web and CLI — heavy web usage reduces CLI availability.
• Bailian Coding Plan is personal use, coding tools only — not suitable for backend APIs or automated scripts.
• Qoder CLI credits are monthly and non-rolling — unused credits do not carry over.
• Chinese model pricing is subject to change (the market has seen both price wars and 50–80% increases for some providers). Always verify at Bailian pricing page.
• SWE-bench scores are not directly comparable across vendors: Opus 4.7's 87.6% is on SWE-bench Verified (Anthropic-reported); Qwen3.7-Max's ~80% SWE-V / 60.6% Pro is from the Qwen blog using an agent harness; DeepSeek V4 Pro numbers come from the DeepSeek technical report. Harness and scaffolding materially affect scores. • Quality-cost trade-off: cheaper models (Qwen3.7-Plus, DeepSeek V4 Flash) may require more iterations for complex tasks, partially offsetting token savings.
Recommendation for ~100-member Singapore team: Start with Option D (Qoder CLI Pro+) for heavy coding users and Pro for others — roughly 3× cheaper per developer with an integrated multi-agent experience and minimal setup. For teams wanting maximum savings with full cost control, Option A (OpenCode + Bailian pay-go with Qwen3.7-Max) delivers near-frontier quality at ~3–4× lower cost. Consider a hybrid approach (Option F) during transition, keeping Claude Code for critical paths while onboarding Bailian.
