Long-Horizon Benchmark

This benchmark is not a generic coding leaderboard. It measures whether a replacement model can preserve the same SDLC workflow quality that an existing Claude-based skill system already delivers in production.

The production Claude artifacts are treated as the baseline, not perfect truth. GLM output is judged against the original requirement context, the stage skill, the prior artifacts, and the accepted baseline.

Core Score Model

30%

Context Retention

Preserves requirements, constraints, decisions, and assumptions across stages.

30%

Stage Completion

Produces the expected deliverable for each stage well enough to move forward.

25%

Workflow Continuity

Moves the project forward coherently without drifting, restarting, or contradicting earlier work.

15%

Reviewability

Makes outputs clear enough for a human to review, approve, or correct efficiently.
